# Zappy — Sito ufficiale

Sito statico (HTML + CSS + JS) per Zappy, sistema di ordini e pagamenti al tavolo per bar, pub e ristoranti.

## Struttura

- `index.html` — Home
- `ecosistema.html` — Ecosistema prodotto
- `come-funziona.html` — Flusso operativo
- `perche-zappy.html` — Valore e missione
- `prezzi.html` — Pacchetti e prezzi
- `contatti.html` — Form contatti (invio su WhatsApp)
- `css/style.css` — Design system
- `js/main.js` — Interazioni e animazioni
- `images/` — Asset grafici e logo

## Sviluppo locale

Sito statico: basta aprire `index.html` con un browser, oppure servirlo con un server statico.
